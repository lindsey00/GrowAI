## 🎯 핵심 특징

### 1. 3줄 구조 ✅

```
1줄: 데이터로 진단하고, 최적의 AI/로봇 솔루션을 매칭하며,
     - text-cyan-400/90
     - font-semibold
     - 기능 설명

2줄: 도입 후의 경제적 이익(ROI)과 ESG 성과를 시뮬레이션하는
     - text-cyan-400/90
     - font-semibold
     - 가치 제안

3줄: 전문가 증강형(Expert Augmentation) 플랫폼
     - text-cyan-400
     - font-bold
     - 핵심 정체성
```

### 2. 스타일 통일성 ✅

```css
/* 컨테이너 */
text-xl md:text-2xl          /* 크기 (반응형) */
text-gray-300                /* 기본 색상 */
font-medium                  /* 중간 굵기 */
max-w-5xl                    /* 최대 너비 */
leading-relaxed              /* 줄 간격 */

/* 1-2줄 */
text-cyan-400/90             /* 시안 90% */
font-semibold                /* 굵기 600 */

/* 3줄 (강조) */
text-cyan-400                /* 시안 100% */
font-bold                    /* 굵기 700 */
```

### 3. 계층 구조 ✅

```
제목 (IS HERE):
- text-7xl md:text-9xl
- font-black
- 최대 강조

부제목 (플랫폼 설명):
- text-xl md:text-2xl
- font-medium/semibold/bold
- 중간 강조

버튼:
- text-lg
- font-black
- 행동 유도
```

## 📐 크기 비율

### 제목 vs 부제목

```
제목:     text-7xl (72px) → text-9xl (128px)
부제목:   text-xl (20px) → text-2xl (24px)

비율: 약 6:1 (desktop)

→ 명확한 시각적 계층
→ 제목 우선, 부제목 보조
```
