## 🎯 사용자 경험

### 초기 상태

```
1. 페이지 로드
2. 좌하단에 플로팅 버튼 표시
3. 빨간 알림 점 애니메이션
```

### 챗봇 열기

```
1. 플로팅 버튼 클릭
2. 챗봇 창 펼쳐짐 (애니메이션)
3. 초기 환영 메시지 표시
```

### 메시지 전송

```
1. 질문 입력
2. Enter 또는 전송 버튼 클릭
3. 사용자 메시지 표시
4. 타이핑 인디케이터 (1.5초)
5. 봇 응답 + 출처 표시
6. 자동 스크롤
```

### 최소화

```
1. 최소화 버튼 클릭
2. 헤더만 표시 (64px 높이)
3. 메시지 영역 숨김
```

### 닫기

```
1. X 버튼 클릭
2. 챗봇 창 닫힘
3. 플로팅 버튼으로 복귀
```
