## 🎯 핵심 특징

### 1. MAP (강조) ✅

```css
text-2xl                     /* 크기 24px */
font-black                   /* 굵기 900 */
tracking-tight               /* 타이트 자간 */
bg-clip-text                 /* 텍스트 클립 */
text-transparent             /* 투명 */
bg-gradient-to-r             /* 그라데이션 */
from-blue-400 via-cyan-400 to-blue-500

→ 로고와 동일한 그라데이션
→ 강력한 시각적 임팩트
```

### 2. Manufacturing Analysis & Prediction (우아함) ✅

```css
text-xs                      /* 크기 12px */
font-light                   /* 굵기 300 */
tracking-wide                /* 넓은 자간 */
text-cyan-400/60             /* 시안 60% */
uppercase                    /* 대문자 */

→ 부드럽고 우아한 느낌
→ MAP을 보조하는 역할
```

### 3. 테두리 제거 ✅

```
Before: border + rounded-full + padding
After: 테두리 없음, 순수 텍스트

→ 깔끔하고 모던한 디자인
→ 로고와 완벽한 통일성
```
