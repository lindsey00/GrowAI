## 🐛 문제 해결

### 3D 씬이 안 보이는 경우

```
1. 정상 동작: 폴백 이미지 자동 표시
2. 콘솔 확인: "Spline failed to load, using fallback"
3. 네트워크 확인: Spline CDN 접근 가능 여부
```

### 로딩이 느린 경우

```
1. 로딩 인디케이터 표시 확인
2. 2-3초 대기
3. 폴백 이미지로 전환 (자동)
```

### 애니메이션이 끊기는 경우

```
1. 하드웨어 가속 활성화
   - Chrome: chrome://flags/#ignore-gpu-blocklist
2. GPU 드라이버 업데이트
3. 다른 탭 닫기
```

### 모바일에서 느린 경우

```
1. 자동 최적화 확인
   - 3D 씬 비활성화 (모바일)
   - 파티클 수 감소
2. 정상 동작: 이미지만 표시
```
