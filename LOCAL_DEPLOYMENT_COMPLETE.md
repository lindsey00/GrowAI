## 🐛 문제 발생 시

### 3D 씬이 안 보이는 경우

```

3. 개발자 도구 확인 (F12 → Console)
```

### 애니메이션이 안 보이는 경우

```
1. 천천히 스크롤

```

```
1. 하드웨어 가속 활성화

```
