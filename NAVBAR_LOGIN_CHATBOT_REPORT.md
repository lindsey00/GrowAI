## 📊 성능 최적화

### 상태 관리

```typescript
// Navbar
- isLoginModalOpen: 모달 열기/닫기
- isUserMenuOpen: 사용자 메뉴 열기/닫기
- isLoggedIn: 로그인 상태
- isMobileMenuOpen: 모바일 메뉴 열기/닫기

// LoginModal
- mode: 'login' | 'signup'
- showPassword: 비밀번호 표시/숨김
- isLoading: 로딩 상태
- error: 에러 메시지
- success: 성공 상태

// ChatBot
- isOpen: 챗봇 열기/닫기
- isMinimized: 최소화 상태
- messages: 메시지 배열
- inputValue: 입력 값
- isTyping: 타이핑 상태
```

