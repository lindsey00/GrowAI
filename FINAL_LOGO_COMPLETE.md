```css

/* 호버 상태 */

글로우: 0 0 25px cyan (강력한 시안 글로우)
밝기: 110% (brightness-110)
```

### 애니메이션 세부사항

```
1. 확대 효과
   - scale-125 (25% 확대)
   - 이전: 10% → 현재: 25%

2. 글로우 효과
   - drop-shadow: 0 0 25px rgba(6,182,212,0.8)
   - 강력한 시안 글로우

3. 밝기 증가
   - brightness-110
   - 10% 밝아짐

4. 배경 그라데이션
   - opacity: 30% (이전 20%)
   - blur-2xl (이전 blur-xl)
   - animate-pulse (펄스 애니메이션)

5. 전환 시간
   - duration-700 (이전 500ms)
   - ease-out (부드러운 감속)
```
