- [ ] `float` - 부유 효과
- [ ] `pulse-glow` - 빛나는 효과
- [ ] `shimmer` - 반짝이는 효과

## 🔧 문제 해결 가이드

### 서버가 시작되지 않는 경우

```bash
# 의존성 재설치
npm install

# 캐시 삭제 후 재시작
npm run dev -- --force
```

### 포트 충돌 시

```bash
# 다른 포트로 실행

```

### 빌드 에러 시

```bash

npx tsc --noEmit

# 빌드 테스트
npm run build
```
